## О проекте

> **От динамического хаоса к строгой архитектуре: безопасный код на TypeScript**

Этот репозиторий предназначен для перехода с Vanilla JS на TypeScript. Основной упор сделан на понимание системы типов, создание интерфейсов, работу с классами и настройку конфигурации компилятора (`tsconfig`). Проекты демонстрируют, как типизация помогает избегать ошибок еще на этапе написания кода.

## Быстрый старт

### Установка зависимостей и запуск

```bash
  # 1. Клонирование репозитория
  git clone <repository-url>
  cd -TypeScript_Education-

  # 2. Установка библиотек (TS, ts-node, types)
  pnpm install

  # 3. Запуск главного файла
  pnpm dev
```
